## Support & Troubleshooting

### Issue: Animations not showing
**Solution**: Check that CSS file is imported in Skills.jsx
```javascript
import '../styles/Skills.css';
```

### Issue: Cards not expanding
**Solution**: Ensure onClick handler is on `.category-header` element

### Issue: Scroll animations not working
**Solution**: Verify `containerRef` is attached to section element

### Issue: Theme not switching colors
**Solution**: Check that `html[data-theme="light"]` CSS exists

### Issue: Performance problems
**Solution**: See CUSTOMIZATION_GUIDE.md Section 12 (Performance Optimization)

**Full troubleshooting**: See CUSTOMIZATION_GUIDE.md Section 14
