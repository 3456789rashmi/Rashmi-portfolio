## 🚀 Getting Started

### **Step 1: Install** (5 minutes)

```bash
cd "d:\downloads archive\Portfolio"
npm run install-all
```

### **Step 2: Run** (2 minutes)

```bash
npm run dev
```

Open:

- Frontend: http://localhost:3000
- Backend: http://localhost:5000

### **Step 3: Customize** (30 minutes)

Edit these files in this order:

1. `client/src/pages/Home.js` - Your name & bio
2. `client/src/pages/Projects.js` - Your projects
3. `client/src/components/Skills.js` - Your skills
4. `client/src/components/Footer.js` - Social links
5. `client/src/styles/index.css` - Brand colors

### **Step 4: Test** (10 minutes)

- Visit http://localhost:3000
- Test on mobile view (press F12)
- Check all links
- Test contact form

### **Step 5: Deploy** (varies)

- Push to GitHub
- Deploy frontend (Vercel/Netlify)
- Deploy backend (Railway/Render)

```javascript

  title: 'Project Name',

  image: 'image-url',
  tags: ['React', 'Node.js'],
```

```javascript
{ name: 'React', icon: <FaReact />, level: 90 }
```

```css
:root {
  --primary-color: #YOUR_COLOR;

}
```

### **Content**

- Write clear, compelling project descriptions

- Highlight impact, not just features
- Keep professional tone

- Don't change the dark theme
- Stick with indigo/pink color scheme

- Test on real mobile devices
- Keep it simple and clean

- Update portfolio regularly
